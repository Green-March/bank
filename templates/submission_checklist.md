# Delivery Checklist
Last Updated: YYYY-MM-DD

## Report Quality
- Executive summary includes clear rating and rationale
- Core metrics and trend tables are complete
- Assumptions and scenario boundaries are explicit
- Risks and monitoring points are included

## Data Quality
- Source endpoints and retrieval date are documented
- Unit/currency consistency checked
- Missing values and fallback logic documented

## Output Packaging
- Markdown report generated
- HTML report generated
- Related figures/tables saved
- Reproduction commands documented
