- `validate_value_range()` が最初の period のみから `concept_section` を構築 — Q1/Q3 で CF がスキップされる

**B: market-data-collector 新規作成**

- 同じ認証トークンで利用可能（auth.py 自動更新済み）
- PER/PBR/時価総額/配当利回り/EV の算出が可能に
- 「財務分析レポート」→「投資分析レポート」への質的転換

**C: senior.md ゴースト参照3件修正**
1. `scripts/send_clear_reinit.sh` → 実在しない。正しくは `templates/senior_clear_junior.sh`
2. `queue/review/junior_plan_to_reviewer.yaml` / `reviewer_to_junior_plan.yaml` → 実在しない孤立サブフロー
3. send-keys 3ステップ法（sleep 2 + capture-pane）が CLAUDE.md 2ステップ法（sleep 1）と矛盾

**D: senior_clear_junior.sh 堅牢化**

- Junior番号を正規表現で抽出 → 明示的引数に変更すべき
